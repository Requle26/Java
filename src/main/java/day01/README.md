- Java 프로젝트 생성
- main() 이해
- 변수

### 자료형

|자료형|설명|
|------|----|
|int|정수|
|double|실수|
|String|문자열|
|char|문자 하나|
|boolean|참/거짓|

### 배운 것

- Java는 모든 코드가 클래스 안에 존재한다.
- main()은 프로그램의 시작점이다.
- println()은 출력이고 return은 반환이다.
- String은 큰따옴표, char는 작은따옴표를 사용한다.
